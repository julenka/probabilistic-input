Target Estimation With Particle Filter

The interface below is a simple grid of buttons, one button from the grid can be selected at a time. Clicking a button selects it.

A particle filter is used to estimate the state of the interface (i.e. which button is selected). The update and measure methods for the particle filter are configurable.
