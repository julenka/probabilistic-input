Decaying Kalman filter

    Move the mouse around the screen, the code will measure the position of the 
    mouse, in green, and add artificial uncertainty, in red. The code will use variety 
    of filters (kalman, EWMA) to predict the position, blue. The demo also
    contains several methods of visualizing the uncertainty about the estimation
    for points filtered using a kalman filter.

    Based off of code by ben. Matrix library provided by sylvester.js
    
    'a' - Show/hide ground truth (green)
    's' - Toggle between visualization modes.
    'd' - Change filter [kalman, ewma, none]
    'f' - Clear canvas
