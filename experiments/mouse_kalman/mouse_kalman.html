Decaying Kalman filter

    Move the mouse around the screen, the code will measure the position of the 
    mouse and add artificial uncertainty represented by the red points.  
    The code will use a kalman filter to predict the true position represented 
    by the blue points.

    Unlike an ordinary kalman filter, I've added a decay constant to this code
    which errodes confidence with time by multiplying the P matrix.  This allows
    the filter to adjust to changes in velocity over time.

    A riff off of code by ben. Matrix library provided by sylvester.js
    
    'a' - Show/hide measurements.
    's' - Toggle between visualization modes.
    'd' - Change filter [kalman, ewma, none]
    'f' - Clear canvas
